# Super Proxy

Super Proxy is an open-source AI access-management frontend for small teams.

It presents a single OpenAI-compatible gateway for connecting AI subscriptions and API accounts while managing:

- Individual employee API keys
- Per-model access controls
- Spend limits and usage visibility
- Intelligent routing and provider fallback
- Open-source and self-hosted deployment

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production build

```bash
npm run build
npm run start
```

## Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- shadcn/ui
- Recharts
- Motion / MagicUI animated beams
